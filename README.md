# 🌐 Projeto HTML Básico

## 📌 Descrição

Este projeto foi desenvolvido como parte de um desafio de HTML, com o objetivo de praticar as principais tags da linguagem.

## 🧩 Tecnologias utilizadas

* HTML

## 📚 Aprendizados

* Estrutura básica de um site
* Uso de tags como listas, links e formatação de texto
* Organização de conteúdo em páginas web

## 🚀 Autor

Projeto desenvolvido para fins acadêmicos.
